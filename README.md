# ⚡ Electric Vehicle Data Analysis (Excel)

## 🔍 Problem Statement

Analyze electric vehicle data to understand market trends, pricing patterns, and technological factors affecting EV performance.

## 📁 Dataset

* Source: Kaggle
* Size: 3000+ rows, 17 columns
* Domain: Automotive / Business

## 🛠 Tools & Technologies

* Microsoft Excel
* Data Cleaning
* Statistical Functions
* Charts & Visualization

## 📊 Analysis Performed

* Removed missing values and duplicates
* Outlier detection using IQR method
* Descriptive statistics
* Data visualization (histograms, pie charts, bar charts)

## 📈 Key Insights

* EV prices vary significantly across models
* Battery capacity strongly influences driving range
* Sales distribution is consistent across years
* Market shows stable growth trends

## 💡 Business Impact

* Helps companies understand EV market dynamics
* Supports pricing and product strategy decisions
* Identifies key factors influencing consumer demand

## 📸 Visualizations

* Battery capacity vs range charts
* Price distribution graphs
* Sales trends

## 🚀 Conclusion

This project highlights how Excel can be used effectively for large-scale data analysis and business insights generation.
